## Use in Your Project

Install as a dependency in your TypeScript/JavaScript project:

```bash
npm install @cadit-app/puerto-rico
```

Then import and use it in your code.

## Build Locally

Clone this repo and build 3D files offline:

```bash
git clone https://github.com/CADit-app/puerto-rico.git
cd puerto-rico
npm install
npm run build:3mf
npm run build:glb
```
